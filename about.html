About


This is a GUI based web application that lets us select a folder at any desired location within our local system and gives us an option to select the time/day at which the application should check if any changes were made within the folder and syncs them with the targeted Google Drive Folder.

The Application contains a clean and sophisticated GUI.

It asks users to login or connect to the Google Drive account they want to sync their files into.

It has functionality to select one or more folders individually for sync.

Sync time for each individual folder can be different. So basically a feature that enables the user to provide every folder their individual sync time. (For eg. everyday at 2pm or every monday at 11am)

At the initial sync if a folder with the exact same name is not found in the desired GDrive then a folder with exact same name is created in the drive automatically and everything gets synced there.

Feature to enable and disable sync individually for folders and an added feature which lets a user permanently delete sync for any disabled sync folder. (In case the sync entry is deleted it will not affect the files or folders that were synced previously).
